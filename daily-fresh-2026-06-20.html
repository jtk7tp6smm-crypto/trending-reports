它是什么

给 AI Agent 补充跨平台读取能力的工具集合，统一 9+ 平台（推特/Reddit/YouTube/GitHub/B 站/小红书等）的数据获取接口，让 Agent 自己选择和调用上游。

对你可能有用

Hermes 当前浏览器 + 小红书 + GitHub 三件套能覆盖一部分跨平台调研，但没编排成"一次提问多平台汇总"的路径。Agent-Reach 的增量价值在"把分散能力编排"这件事本身。

需要警惕什么

  • 依赖多个第三方上游，某个失效会影响整体
  • Twitter/Reddit/小红书需登录或 Cookie，账号风控风险存在
  • 安装涉及 Python、Node.js、GitHub CLI、MCP 依赖，不是单文件工具
