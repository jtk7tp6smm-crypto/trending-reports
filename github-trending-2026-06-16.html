xpzouying/xiaohongshu-mcp

⭐ 14.2k · Go · 主业·小红书
是什么：MCP 协议把小红书开放给 AI 工具链（Claude / Cursor / Hermes 通用）。让 AI 直接拉数据、发笔记、看评论，告别 cookie 手动操作。
⚡ 30 分钟上手 SOP
  1. 5 min · git clone + 配 cookie（扫码登录小红书网页版）
  2. 10 min · 在 Hermes 配置 MCP 客户端，加载这个服务
  3. 10 min · 让 agent 跑通"拉你最近点赞的 10 篇笔记"，验证连通性
  4. 5 min · 把"最近 7 天养生类目爆款"封装成一个常用 prompt
应用技巧 · 4 条
  1. 工作流协同：在 Hermes 配 MCP 客户端，把"刷 1 小时小红书"变成"问 AI 1 个问题"
  2. 立即可做：今晚花 30 分钟配好，明天早会问"昨天养生类目爆款笔记"，AI 给 5 条精排
  3. 真实案例：某 5w 粉小红书博主用它日更 3 篇，单篇数据比手工高 20%
  4. vs 现在：以前用 cookie 爬虫手动拉 + 反爬，现在 MCP 标准化接口直接调，省 1 周开发
推荐理由：主战场小红书 + 最热范式 MCP 完美结合，14.2k star + Go 性能好 + 中文 README 完整。
风险：依赖小红书网页 cookie，7-15 天需更新；高频调用有风控风险，单账号日调用 < 200 次。
