主推荐

Panniantong/Agent-Reach

⭐ 36,888 stars+1,575 stars（+4.46%）值得试用评估

它是什么

一个给 AI 助手补充跨平台读取能力的工具集合，统一 9+ 平台（推特/Reddit/YouTube/GitHub/B 站/小红书等）的数据获取接口，让 Agent 自己选择和调用上游工具。

对你可能有用

你最近三个增长方向（直播运营、小红书内容、AI 工具自动化）都需要做跨平台调研。Hermes 当前的浏览器 + 小红书 + GitHub 三件套能覆盖一部分，但没有把这些能力编排成"一次提问多平台汇总"的路径。Agent-Reach 的增量价值在"把分散能力编排"这件事本身，对 GEO 研究和自动化学习也有参考价值。

需要警惕

  • MIT License（GitHub API 已确认）
  • 54.55h +1,575 stars 增长（market-snapshot 跨日 diff 数据，已建立）
  • 必须登录小红书账号（README 明确说明）
  • 同一账号不能多端登录（项目 README 提到，是小红书平台限制）
  • Twitter/Reddit/小红书等渠道需要登录状态或 Cookie
